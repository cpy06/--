# 基于单向链表的学生成绩单管理系统

## 一、项目简介

本程序使用 C 语言编写，采用单向链表组织学生成绩单数据，实现了文件读写、增加、批量增加、修改、删除、批量删除、查询、排序、撤销等功能，适合作为“程序设计基础”课程设计或实验作业。

每条学生记录包含：

- 学号
- 姓名
- 语文成绩
- 数学成绩
- 英语成绩
- C语言成绩
- 数据结构成绩
- 总分
- 平均分

## 二、主要功能

1. 从用户输入的文件路径读取文本成绩单。
2. 将当前数据保存到用户指定文件。
3. 显示全部学生记录。
4. 添加单个学生记录。
5. 批量添加学生记录。
6. 修改学生记录，支持修改学号、姓名和成绩。
7. 删除单个学生记录。
8. 批量删除学生记录。
9. 按学号、姓名、模糊关键字、单科成绩、总分等条件查询。
10. 按学号、总分、单科成绩排序。
11. 撤销上一步误操作。
12. 清空全部记录。

## 三、文件格式说明

程序使用文本文件保存数据，每行一条记录，格式如下：

```text
学号 姓名 语文 数学 英语 C语言 数据结构
```

示例：

```text
2023001 张三 88 90 85 92 87
2023002 李四 76 81 79 84 80
```

项目中提供了示例文件 `data.txt`。

## 四、编译与运行

在当前目录下执行：

```bash
cc main.c -o transcript
./transcript
```

## 五、设计说明

1. 使用 `List` 结构体统一管理链表头指针和节点数量。
2. 排序使用归并排序，适合链表结构，时间复杂度为 `O(n log n)`。
3. 为了提升用户体验，程序加入了输入校验、路径校验和撤销功能。
4. 撤销功能采用“操作前快照”的方式实现，最多保留最近 `20` 次变更记录。
